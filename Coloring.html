Coloring for Complex Functions

An Interactive Applet powered by Sage and MathJax.

(By Kelsey Norman with HTML code from Prof. Gregory V. Bard)

\textrm{This demo displays a coloring of a complex function.} \\ \textrm{In particular, this demo takes a function of one variable (x or z) and outputs the coloring. } \\ \textrm{The modulus is represented by the brightness and the argument is represented by the hue. }
